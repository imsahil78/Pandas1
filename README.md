# 📊 Pandas Practice Notebook

This repository contains my practice notebook for learning and exploring the basics of **Pandas** in Python. It covers important dataframe operations, data cleaning techniques, indexing, grouping, joining, and more.

## 🚀 Topics Covered

- Pandas Series
- DataFrames
- Selecting and Indexing Columns
- Creating New Columns
- Removing Columns and Rows
- Accessing Rows
- Slicing
- Conditional Selection
- Handling Missing Data
- Removing Null Values
- Filling Missing Data
- Merging, Joining, and Concatenation
- GroupBy Operations
- Aggregation Functions
- DataFrame Operations

## 🛠️ Technologies Used

- Python 🐍
- Pandas
- Jupyter Notebook

## 📂 File Included

- `pandas2.ipynb` → Main notebook containing all Pandas practice examples.

## ▶️ How to Run

1. Clone this repository:

```bash
git clone https://github.com/imsahil78/Pandas1.git
```

2. Open the project folder.

3. Install required libraries:

```bash
pip install pandas notebook
```

4. Start Jupyter Notebook:

```bash
jupyter notebook
```

5. Open `pandas2.ipynb` and run the cells.
